# bigwhammystats
Streamlit app to track The Big Whammy league stats, awards, and standings.

## Articles

Articles are Markdown files stored in `articles/`.

To publish a new article:

1. Copy `articles/_template.md`.
2. Rename it using a URL-friendly slug, for example `gw20-review.md`.
3. Update the front matter at the top.
4. Change `status: "draft"` to `status: "published"`.
5. Add optional images under `articles/images/` and reference them from the article.

Suggested categories:

- Gameweek Review
- Awards
- Last Person Standing
- Iron Man
- Announcements
